Remix CSS - Documentation

What is Remix CSS?

Remix CSS is a set of styles that you can use as a foundation for your own stylesheet when starting a new project. The library is meant to be a stylesheet reset with basic styling for common elements. Remix CSS is easily themable using LessCSS variables and includes some utility styles to make finishing your stylesheet easier.

Installation

Install Remix CSS via NPM.

$ npm install --save-dev remix-css


Create an aias to the theme file in your webpack config.

{
  resolve : {
    alias : {
      'theme' : path.resolve(__dirname, 'node_modules/remix-css/src/default.theme.less')
    }
  }
}


Create a new LessCSS stylesheet and import the Remix styles into it.

@import "~theme";
